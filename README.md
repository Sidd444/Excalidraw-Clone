# Excalidraw Clone

## Description

This project is a clone of the Excalidraw website, created using HTML and CSS. It replicates the core features and design of the original site, allowing users to experience how the Excalidraw website works on the client side.

## Demo

You can view a live demo of the project [here](https://sidd444.github.io/Excalidraw-Clone/).

## Features

- **Responsive Design**: The website is designed to be responsive, ensuring an optimal viewing experience on various devices.
